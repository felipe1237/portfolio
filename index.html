Página de cuestionarios

arrow_forward
expand_content collapse_content

Aquí lo que hice fue crear una sencilla pagina en la que se puede crear un cuestionario de preguntas,agregar sus respectivas opciones de respuestas y elegir un tiempo determinado para su resolución. Aquí utilizo,html,css,javascript,nodejs con Express,autenticacion de usuario ,registro con JWT Mysql,cookies de session. Este proyecto cuenta con 3 secciones,el login/registro,la pagina para crear los cuestionarios cuyo acceso solo esta limitado a usuarios de tipo admin,y la pagina de resolucion del custionario
